# MonthlyExpenseTracker
## Monthly Income Expense Tracker
### Technologies
Frontend - React \
Backend - Node.js 

### Setup
#### Backend
- cd backend
- npm install
- npm start

#### Frontend
- cd frontend
- npm install
- npm start

Production Deployment - https://monthly-expense-tracker.vercel.app/
